## 一、核心代码

### 1. 微秒级计时器（TIM3 + 溢出计数）

```c
volatile uint32_t timer3_overflow_count = 0;

void us_TIM(void)
{
    RCC_APB1PeriphClockCmd(RCC_APB1Periph_TIM3, ENABLE);
    TIM_TimeBaseInitTypeDef tim = {0};
    tim.TIM_Prescaler = 71;        // 1MHz = 1μs
    tim.TIM_Period    = 0xFFFF;    // 65ms 溢出一次
    tim.TIM_CounterMode = TIM_CounterMode_Up;
    TIM_TimeBaseInit(TIM3, &tim);
    TIM_ITConfig(TIM3, TIM_IT_Update, ENABLE);
    TIM_Cmd(TIM3, ENABLE);
}

void TIM3_IRQHandler(void)
{
    if (TIM_GetFlagStatus(TIM3, TIM_FLAG_Update) == SET) {
        TIM_ClearFlag(TIM3, TIM_FLAG_Update);
        timer3_overflow_count++;    // 每 65ms 记一次溢出
    }
}

// 获取当前总微秒数（防溢出竞态）
uint32_t micros(void)
{
    uint32_t ovf, cnt;
    do {
        ovf = timer3_overflow_count;
        cnt = TIM3->CNT;
    } while (ovf != timer3_overflow_count);  // 读到一半溢出了？重来
    return (ovf << 16) | cnt;  // 高16位=溢出次数，低16位=CNT
}

void My_Delay_2_us(uint32_t us)
{
    uint32_t start = micros();
    while ((micros() - start) < us);
}
```

### 2. 输入捕获初始化（双通道协同）

```c
void Init_Timebase_InputCapture(void)
{
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_TIM1, ENABLE);

    // 时基：1μs 分辨率，65ms 周期
    TIM_TimeBaseInitTypeDef tim = {0};
    tim.TIM_Prescaler = 71;
    tim.TIM_Period    = 65535;
    TIM_TimeBaseInit(TIM1, &tim);

    // CH1：直接上升沿捕获
    TIM_ICInitTypeDef ic = {0};
    ic.TIM_Channel    = TIM_Channel_1;
    ic.TIM_ICPolarity = TIM_ICPolarity_Rising;
    ic.TIM_ICSelection = TIM_ICSelection_DirectTI;   // 直接
    ic.TIM_ICFilter   = 0x0;
    ic.TIM_ICPrescaler = TIM_ICPSC_DIV1;
    TIM_ICInit(TIM1, &ic);

    // CH2：间接下降沿捕获（同引脚 PA8，捕获高电平结束时刻）
    ic.TIM_Channel    = TIM_Channel_2;
    ic.TIM_ICPolarity = TIM_ICPolarity_Falling;
    ic.TIM_ICSelection = TIM_ICSelection_IndirectTI; // 间接
    TIM_ICInit(TIM1, &ic);

    TIM_ITConfig(TIM1, TIM_IT_CC1 | TIM_IT_CC2, ENABLE);
    TIM_Cmd(TIM1, ENABLE);
}
```

### 3. 捕获中断（只设标志位）

```c
void TIM1_CC_IRQHandler(void)
{
    if (TIM_GetFlagStatus(TIM1, TIM_FLAG_CC1) == SET) {
        TIM_ClearFlag(TIM1, TIM_FLAG_CC1);
        capture_state |= 0x01;   // 上升沿捕获到位
    }
    if (TIM_GetFlagStatus(TIM1, TIM_FLAG_CC2) == SET) {
        TIM_ClearFlag(TIM1, TIM_FLAG_CC2);
        capture_state |= 0x02;   // 下降沿捕获到位
    }
    if (capture_state == 0x03) { // 双沿都到了
        Rang_State = RANG_OK;
        capture_state = 0;
    }
}
```

### 4. 超声波测距状态机

```c
typedef enum { TRIG_SEND, WAIT_ECHO, PROCESS, INTERVAL } state_t;
state_t state = TRIG_SEND;

void ultrasonic_ranging_state_machine(void)
{
    switch (state)
    {
    case TRIG_SEND:
        TIM_SetCounter(TIM1, 0);
        TIM_ClearFlag(TIM1, TIM_FLAG_CC1 | TIM_FLAG_CC2);
        Rang_State = RANG_NO;
        capture_state = 0;

        GPIO_WriteBit(GPIOA, GPIO_Pin_1, Bit_SET);  // Trig 拉高
        My_Delay_2_us(10);                           // 10μs
        GPIO_WriteBit(GPIOA, GPIO_Pin_1, Bit_RESET);

        start_time = currentTick;
        state = WAIT_ECHO;
        break;

    case WAIT_ECHO:
        if (Rang_State == RANG_OK)      { state = PROCESS; break; }
        if (currentTick - start_time > 30) { state = INTERVAL; }  // 30ms 超时
        break;

    case PROCESS:
        ccr1 = TIM_GetCapture1(TIM1);
        ccr2 = TIM_GetCapture2(TIM1);
        // 回绕安全计算
        uint32_t width = (ccr2 >= ccr1)
            ? (ccr2 - ccr1)
            : (0xFFFF - ccr1 + ccr2 + 1);  // us

        if (width > 100 && width < 20000) {  // 1.7cm ~ 3.4m
            float dist = (width * 1.0e-6f * 340.0f) / 2.0f;
            printf("%.3f m\r\n", dist);
            LED_MODE = LED_ON;              // 测到距离亮灯 0.5s
        }
        start_time = currentTick;
        state = INTERVAL;
        break;

    case INTERVAL:
        TIM_Cmd(TIM1, DISABLE);             // 空闲关定时器省电
        if (currentTick - start_time > 2000) // 2s 间隔
            state = TRIG_SEND;
        break;
    }
}
```

## 二、核心知识点

### 1. 输入捕获信号通路

```
ECHO 引脚 → 输入滤波(去毛刺) → 边沿检测 → 信号选择(直接/间接/TRC)
         → 分频器 → 锁存 CNT 到 CCRx → 触发中断
```

### 2. 双通道协同原理

| 通道 | 边沿 | 选择 | 作用 |
|------|------|------|------|
| CH1 | 上升沿 | 直接 | 锁存高电平开始时刻 → ccr1 |
| CH2 | 下降沿 | 间接 | 锁存高电平结束时刻 → ccr2 |

> 同引脚 PA8，一次完整脉冲得到两个时间戳，`ccr2 - ccr1` = 高电平宽度

### 3. 回绕计算的 int 提升陷阱

```c
// ❌ 危险：uint16_t 运算前隐式提升为有符号 int
uint32_t width = ccr2 - ccr1;  // ccr2 < ccr1 时得负数，转 u32 变巨大值

// ✅ 法一：显式判断回绕
uint32_t width = (ccr2 >= ccr1) ? (ccr2 - ccr1) : (0xFFFF - ccr1 + ccr2 + 1);

// ✅ 法二：16 位模运算自动修正
uint32_t width = (uint16_t)(ccr2 - ccr1);
```

### 4. 超声波测距公式

```
距离 = (高电平时间 × 声速) / 2
     = (width μs × 10⁻⁶ × 340 m/s) / 2
```

> `/2` 是因为声波往返，单程距离需除以 2

### 5. 状态机重构 — 从混乱到清晰

```
修改前：while(1) { 发脉冲 → 死等 → 判标志 → 算距离 → 延时 }  // 逻辑纠缠

修改后：
TRIG_SEND → WAIT_ECHO → PROCESS → INTERVAL
    ↑                                  │
    └──────────── 2s 后 ───────────────┘
```

每个状态只做一件事，通过 `switch-case` 依次切换。

### 6. 中断优先级分配

| 定时器 | 抢占优先级 | 作用 |
|--------|-----------|------|
| TIM2 | 0（最高） | ms 系统时钟，绝不可丢 |
| TIM1 CC | 1 | 输入捕获 |
| TIM3 | 3（最低） | μs 溢出计数 |

### 7. 微秒计数器 do-while 防竞态

```c
do {
    ovf = timer3_overflow_count;  // 读高16位
    cnt = TIM3->CNT;              // 读低16位
} while (ovf != timer3_overflow_count);
// 如果读取期间刚好溢出，ovf 不匹配 → 重读
```

## 三、心得

- 输入捕获 + 状态机是经典组合：**中断设标志，主循环走状态**
- 回绕计算是 16 位计数器最隐蔽的坑——C 语言的整型提升规则在嵌入式里是真会出 bug
- 状态机重构花了 4 小时，但这是"能用→可维护"的质变——先画流程图再写代码是铁律
- 微秒级计时器（TIM3+溢出计数）是后续所有精密时序的基石
