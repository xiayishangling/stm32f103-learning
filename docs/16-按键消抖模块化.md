# 6.19 按键消抖模块化封装

> **芯片**：STM32F103C8T6 | **环境**：CubeMX + VSCode + Keil  
> **主题**：按键消抖状态机独立封装（key_debounce_t）、回调函数注册、osKernelGetTickCount

> 🔗 衔接 6.18：6.18 中按键消抖逻辑内嵌在 `vIP_KEYTask` 里；6.19 将其抽象为独立模块 `My_Code2.h/.c`，通过回调函数与业务解耦，可直接替换 6.18 中的消抖代码。

## 一、完整代码

### My_Code2.h

```c
#ifndef INC_MY_CODE2_H_
#define INC_MY_CODE2_H_

#include "My_Code1.h"

#define KEY_DEBOUCNE_TIME_MS 20

// 按键消抖回调
typedef void(*key_callback_t)(void);

// 按键消抖状态机
typedef enum {
    key_idle,              // 按键空闲
    key_press_debounce,    // 按下消抖
    key_pressed,           // 确认按下
    key_release_debounce   // 释放消抖
} KEY_DeBounce_StateMachine;

// 按键消抖状态机封装
typedef struct {
    GPIO_TypeDef *port;
    uint16_t pin;
    KEY_DeBounce_StateMachine key_debounce_state;
    uint32_t key_debounce_tick;     // 按键消抖专用时间
    key_callback_t on_press;        // 按下回调
} key_debounce_t;

// 全局按键实例
extern key_debounce_t key1; // 按键1
extern key_debounce_t key2; // 按键2

// 按键消抖状态机函数
void KEY_DeBounce_State_Machine(key_debounce_t *key);

// 按键消抖确认按下回调函数
void KEY_SetPressCallback(key_debounce_t *key, key_callback_t callback);

// 回调函数：按下按钮发生的操作
void KEY1_Callback(void);
void KEY2_Callback(void);

// 初始化按键消抖
void KEY_DeBounce_Init(void);

#endif /* INC_MY_CODE2_H_ */
```

### My_Code2.c

```c
#include "My_Code2.h"

// 初始化按键1实例
key_debounce_t key1 = {
    .port = GPIOA,
    .pin = GPIO_PIN_1,
    .key_debounce_state = key_idle,
    .key_debounce_tick = 0,
    .on_press = NULL
};

// 初始化按键2实例
key_debounce_t key2 = {
    .port = GPIOA,
    .pin = GPIO_PIN_2,
    .key_debounce_state = key_idle,
    .key_debounce_tick = 0,
    .on_press = NULL
};

void KEY_DeBounce_State_Machine(key_debounce_t *key)
{
    uint8_t key_level = HAL_GPIO_ReadPin(key->port, key->pin);

    switch (key->key_debounce_state)
    {
    case key_idle:
        if (key_level == GPIO_PIN_RESET) {
            key->key_debounce_tick = osKernelGetTickCount();  // 1 tick = 1ms
            key->key_debounce_state = key_press_debounce;
        }
        break;

    case key_press_debounce:
        if (key_level != GPIO_PIN_RESET) {
            key->key_debounce_state = key_idle;
        } else {
            if (osKernelGetTickCount() - key->key_debounce_tick > KEY_DEBOUCNE_TIME_MS) {
                key->key_debounce_state = key_pressed;
                if (key->on_press) key->on_press();  // 确认按下 → 回调
            }
        }
        break;

    case key_pressed:
        if (key_level != GPIO_PIN_RESET) {
            key->key_debounce_tick = osKernelGetTickCount();
            key->key_debounce_state = key_release_debounce;
        }
        break;

    case key_release_debounce:
        if (key_level == GPIO_PIN_RESET) {
            key->key_debounce_state = key_pressed;
        } else if (key_level != GPIO_PIN_RESET) {
            if (osKernelGetTickCount() - key->key_debounce_tick > KEY_DEBOUCNE_TIME_MS) {
                key->key_debounce_state = key_idle;
            }
        }
        break;
    }
}

void KEY_SetPressCallback(key_debounce_t *key, key_callback_t callback)
{
    key->on_press = callback;
}

// 使用按键回调时，放到最上面初始化
void KEY_DeBounce_Init(void)
{
    KEY_SetPressCallback(&key1, KEY1_Callback);
    KEY_SetPressCallback(&key2, KEY2_Callback);
}

// 回调函数：按下按钮发生的操作
void KEY1_Callback(void)
{
    // 示例：如传递 keymessage
    // KEYMessage msg = {.w25q64_state = W25Q64_Save_State};
    // osMessageQueuePut(KEY_QueueHandle, &msg, 0, 0);
}

void KEY2_Callback(void)
{
```

## 二、要点

### 1. 与 6.18 的联系

| 6.18（内嵌版） | 6.19（模块化版） |
|----------------|-----------------|
| 消抖逻辑写在 `vIP_KEYTask` 里 | 抽象为 `key_debounce_t` 结构体 |
| 按键实例靠局部变量 | 全局结构体 + 指定初始化器 |
| 用 `HAL_GPIO_ReadPin` 直读 | 状态机封装，主循环调 `KEY_DeBounce_State_Machine()` |
| 回调未独立 | `on_press` 函数指针注册回调 |

### 2. 新变化

- **`osKernelGetTickCount()`**：替代裸机的 `GetTick()`，FreeRTOS 原生 tick 计数
- **指定初始化器**：`{ .port = GPIOA, .pin = GPIO_PIN_1, ... }` 更清晰
- **回调模板留空**：`KEY1_Callback` / `KEY2_Callback` 留待业务填充（如 6.18 的队列发送）

### 3. 使用方式

```c
// 初始化
KEY_DeBounce_Init();

// 主循环或任务中定时调用
KEY_DeBounce_State_Machine(&key1);
KEY_DeBounce_State_Machine(&key2);
```
